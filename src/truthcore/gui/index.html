API Endpoints

  • GET /health
    Health check - returns server status and version
  • GET /api/v1/status
    Get server capabilities and configuration
  • POST /api/v1/judge
    Run readiness check with specified profile and options
  • POST /api/v1/intel
    Run intelligence analysis on historical data
  • POST /api/v1/explain
    Explain invariant rule evaluation
  • GET /api/v1/cache/stats
    Get cache statistics
  • POST /api/v1/cache/clear
    Clear all cache entries
